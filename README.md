# RuneForge Studio: Vel'Nathis & The Shard

A self-contained psychological horror story engine for D&D 5e (Foundry VTT v12+, dnd5e v5.3.x).

## Contents

- **Vel'Nathis** — Tier B NPC, Tiny Fiend (Demon), CR 4, Neutral Evil. A 140-year-old bound demon who communicates through dreams. Full statblock, features, biography, plot hooks, secrets, and escalation track.
- **Shard of Vel'Nathis** — Very Rare cursed loot item. DC 14 WIS save each long rest when within 10 ft. Fully mechanically active with 1/dawn use and Whisper of Vel'Nathis save activity.
- **Vel'Nathis Compulsion Table** — 1d10 RollTable with full mechanical consequences for each result, playable directly from the compendium.
- **GM Journal** — 5-page structured guide: Introduction, NPC Reference (with embedded statblock link and portrait), Item Reference (with workflow), Compulsion Table Reference (with escalation guidance), and full Lore & Story.

## Installation

**Via manifest URL (free/public release):**
1. In Foundry VTT → Add-on Modules → Install Module
2. Paste manifest URL:
   `https://github.com/RuneForgeStudio/runeforgestudio-velnathis/releases/latest/download/module.json`

**Manual install:**
1. Download `module.zip` from the latest release
2. Extract into your Foundry `Data/modules/` folder
3. The folder must be named `runeforgestudio-velnathis`

## Usage

1. Enable the module in your world
2. Import content from the compendium packs
3. Drop the **Shard of Vel'Nathis** item into the world as treasure
4. Open the **GM Journal** and read the Introduction page
5. When a character sleeps within 10 ft. of the shard, enable the Curse Active Effect and begin triggering nightly WIS saves

Full workflow documented in the GM Journal → Item Reference page.

## Compatibility

- Foundry VTT: v12+ (verified v14)
- D&D 5e system: v3.3+ (verified v5.3)
